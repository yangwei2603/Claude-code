## 二、CI执行率指标

### 2.1 CI点执行率

| 指标 | 字段 | 单位 | 说明 |
|------|------|------|------|
| CI点执行率 | CI_POINT_RATE | % | 偏差≤7节的点占比，反映速度控制精度 |
| 小速度占比 | NO_CI_SLOW_RATE | % | 实际速度低于CI推荐>7节的占比 |
| 大速度占比 | NO_CI_FAST_RATE | % | 实际速度高于CI推荐>7节的占比 |

**计算公式：**
```
CI点执行率 = COUNT(|PEP_CAS - QAR_IAS_C| ≤ 7) / COUNT(总点数)
小速度占比 = COUNT(QAR_IAS_C - PEP_CAS > 7) / COUNT(超限总点)
大速度占比 = COUNT(PEP_CAS - QAR_IAS_C > 7) / COUNT(超限总点)
```

**数据来源：** `T_ADS_FOQA_IFP_CI_ANYS` 表，PEP_CAS=CI推荐真空速，QAR_IAS_C=QAR校正指示空速

### 2.2 CI航班执行率

| 指标 | 字段 | 单位 | 说明 |
|------|------|------|------|
| 公司CI执行率 | CI_EXECUTE_RATE | % | CI点执行率≥60%的航班占比 |
| 国内CI执行率 | CI_EXECUTE_RATE_RG | % | DORI=0的航班CI执行率 |
| 国际及区域CI执行率 | CI_EXECUTE_RATE_AREA | % | DORI=1的航班CI执行率 |

**计算公式：**
```
CI航班执行率 = COUNT(CI点执行率 ≥ 0.6) / COUNT(总航班)
```

## 八、指标计算汇总

### 8.1 油耗差异计算

```sql
-- 轮挡耗油差异（实际 - 计划）
CHOCK_FUEL_DIFF = CHOCK_FUEL - PLAN_CHOCK_FUEL

-- 轮挡耗油差异率
CHOCK_FUEL_DIFF_RATE = (CHOCK_FUEL - PLAN_CHOCK_FUEL) / PLAN_CHOCK_FUEL

-- 空中耗油差异
AIRBORNE_FUEL_DIFF = AIRBORNE_FUEL - PLAN_AIRBORNE_FUEL
```

### 8.2 效率指标计算

```sql
-- 轮挡小时耗油
CHOCK_FUEL_H = SUM(CHOCK_FUEL) / (SUM(CHOCK_TIME) / 60)

-- 空中小时耗油
AIRBORNE_FUEL_H = SUM(AIRBORNE_FUEL) / (SUM(AIRBORNE_TIME) / 60)
```

### 8.3 CI执行率计算

```sql
-- 从QAR数据计算单航班CI点执行率
SELECT
    LEG_ID,
    CASE WHEN COUNT(PEP_CAS) > 0 THEN
        ROUND(COUNT(CASE WHEN ABS(PEP_CAS - QAR_IAS_C) <= 7 THEN 1 END)
              / COUNT(PEP_CAS), 4)
    ELSE NULL END AS CI_POINT_RATE
FROM ADS.T_ADS_FOQA_IFP_CI_ANYS
GROUP BY LEG_ID

-- 从航班明细表计算CI航班执行率
SELECT
    ROUND(SUM(CASE WHEN CI_POINT_RATE >= 0.6 THEN 1 ELSE 0 END) / COUNT(*), 4) AS CI_EXECUTE_RATE
FROM ADS.T_ADS_SFUEL_STAT_DTL
```

## 九、指标优先级

| 优先级 | 指标 | 说明 |
|--------|------|------|
| P0 | CI_EXECUTE_RATE | CI执行率，核心节油指标 |
| P0 | CHOCK_FUEL_H | 轮挡小时耗油，地面效率核心 |
| P0 | AIRBORNE_FUEL_H | 空中小时耗油，飞行效率核心 |
| P1 | RTN_ALT_RATE | 返航备降率，影响油耗和航班正常性 |
| P1 | DIRECT_RATE | 直飞率，直飞节省油耗和时间 |
| P2 | FLT_ALTITUDE_RATE | 高度执行率，影响油耗 |
| P2 | SINGLE_TAXI_RATE | 单发滑行率，地面滑行效率 |
