# 财务分析 Skill（航司数据专用）

## 能力范围

- 六大航司经营对比分析（国航、东航、南航、海航、春秋、吉祥）
- CASK 六维成本分解（航油、维修、人力、起降、折旧、营销）
- 春秋航空低成本模式专项分析
- 年报/年报数据获取与结构化处理
- 航司盈利能力与运营效率评估
- 竞争壁垒分析与战略建议

## 分析流程

### 1. 数据获取

使用 `finance-data-retrieval` skill 获取财务数据：

- 营业收入、净利润、CASK、ASK、RPK
- 航油成本占比、维修成本、人力成本
- 客座率、飞机利用率
- 六大航司股票代码：

| 航司 | 股票代码 |
|------|----------|
| 中国国航 | 601111.SH |
| 东方航空 | 600115.SH |
| 南方航空 | 600029.SH |
| 海南航空 | 600221.SH |
| 春秋航空 | 601021.SH |
| 吉祥航空 | 603885.SH |

### 2. CASK 六维分解

```
CASK = 航油成本/ASK + 维修成本/ASK + 人力成本/ASK
     + 起降费/ASK + 折旧摊销/ASK + 其他成本/ASK
```

低成本航司（春秋）vs 全服务航司的分解对比重点：

- 航油成本占比（LCC 低约 2-3pp）
- 人力成本效率（ASK/员工）
- 座公里收益（RASK）与客座率的平衡

### 3. 春秋成本专项下钻

春秋低成本壁垒分析维度：

- **机队利用率**：日利用率 vs 行业均值
- **单位成本**：CASK 行业最低，对标 Southwest
- **两舱 vs 经济舱**：单一舱位配置策略
- **维修策略**：自修比例 vs 外包成本对比
- **人力效率**：飞行员/乘务员人均 ASK

### 4. 风险矩阵

| 风险类型 | 影响程度 | 发生概率 | 应对策略 |
|----------|----------|----------|----------|
| 航油价格波动 | 高 | 高 | 套保比例建议 |
| 汇率风险 | 高 | 中 | 美元负债敞口 |
| 竞争加剧 | 中 | 高 | 差异化策略 |
| 宏观需求 | 高 | 中 | 运力调配 |

### 5. 输出格式

- **Markdown 报告**：结构化分析文档，适合存档
- **PPT 演示文稿**：适合向管理层汇报，使用 `pptx` skill

## 参考数据源

- 年报/年报（PDF）→ 提取关键财务指标
- neodata-financial-search / finance-data-retrieval → 实时行情与财务数据
- 历史 CASK 对比（近 3 年趋势）

## 注意事项

- 航油成本受国际油价影响，季节性波动明显，分析时标注时间维度
- 汇率风险：人民币兑美元波动对美元负债航司（国航）影响更大
- 低成本航司（LCC）分析时，关注单位 ASK 非航空收入贡献
