# 同花顺API期货交易信号系统 - 原型

## 项目概述

基于同花顺API技术需求文档（GF-QH-2026-001）构建的模拟原型系统，用于验证期货自动化交易信号逻辑。

**注意**：由于没有真实的同花顺API访问权限，本系统使用模拟数据进行原型验证。

## 项目结构

```
projects/hyt/
├── CLAUDE.md           # 本文档
├── config.py           # 配置文件
├── main.py             # 入口脚本
├── src/
│   ├── __init__.py
│   ├── models.py       # 数据模型
│   ├── market_data.py  # 行情数据处理
│   ├── positions.py    # 会员持仓数据处理
│   ├── signals.py      # 信号生成引擎
│   ├── exporter.py     # CSV/Excel导出
│   └── simulator.py    # 模拟数据生成器
└── tests/
    ├── __init__.py
    └── test_signals.py # 信号逻辑测试
```

## 核心信号逻辑

### 做多信号（条件全部满足）
| 条件 | 说明 |
|------|------|
| 1 | 当日成交量 > 前一日成交量 |
| 2 | 当日持仓量 > 前一日持仓量 |
| 3 | 当日收盘价 > 前一日收盘价 |
| 4 | 目标席位多头持仓增加 > 2000手 |

### 平多信号（条件全部满足）
| 条件 | 说明 |
|------|------|
| 1 | 当日成交量 > 前一日成交量 |
| 2 | 当日持仓量 < 前一日持仓量 |
| 3 | 当日收盘价 < 前一日收盘价 |
| 4 | 目标席位多头持仓下降 > 2000手 |
