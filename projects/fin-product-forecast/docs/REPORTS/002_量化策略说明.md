# FX Quant 策略脚本使用说明（先模拟盘）

> 该脚本用于**研究与回测**，不应直接当作实盘自动交易系统。

## 文件
- `tools/fx_quant_strategy.py`：汇率量化回测脚本
- 输入：CSV（字段 `timestamp, close`，可选 `spread_bps`）
- 输出：逐日信号与净值曲线 CSV

## 快速运行
```bash
python3 tools/fx_quant_strategy.py \
  --csv reports/sample_usdcny.csv \
  --out reports/fx_signals_sample.csv \
  --lookback 20 \
  --vol-window 30 \
  --target-vol 0.10 \
  --max-leverage 2.0 \
  --stop-loss 0.02
```

## 核心风控机制
1. **波动率目标仓位**：使用滚动波动率缩放仓位，避免高波动期间过度暴露。
2. **杠杆上限**：`--max-leverage` 硬限制绝对仓位。
3. **交易成本惩罚**：按仓位变化（turnover）乘以点差（bps）扣减收益。
4. **单期止损封顶**：`--stop-loss` 对单期净收益做下限保护。

## 建议上线流程
1. 先接入真实历史数据（至少 3-5 年，含不同宏观周期）。
2. 做滚动窗口 walk-forward 验证，避免参数过拟合。
3. 做交易时段、节假日、极端行情压力测试。
4. 模拟盘运行 2-4 周，验证成交滑点与实盘一致性。
5. 达到风险阈值后再小资金试运行。
