# 金融产品预测预警系统

> 为交易决策提供科学依据的量化研究与预警平台

## 项目定位

本系统面向金融产品（汇率、商品、股票等）的价格预测与风险预警需求，核心能力包括：

- **数据采集**：多源官方金融数据接入（FRED、ECB、Bank of Canada 等）
- **预测建模**：监督学习（趋势预测）+ 无监督学习（市场状态聚类）
- **量化回测**：波动率目标仓位、杠杆管理、交易成本建模
- **预警机制**：自定义阈值触发、飞书消息推送
- **报告输出**：可追溯的模型评估报告与策略分析

## 核心模块

| 模块 | 说明 |
|------|------|
| `fx_system/` | 汇率预测前后端系统（HTTP API + HTML 可视化） |
| `tools/` | 量化工具集（回测脚本、代码审计、策略分析） |
| `reports/` | 模型评估与策略分析报告 |
| `agents/` | AI Agent 配置与多会话管理 |
| `workflows/` | 工作流编排配置 |
| `docs/` | 系统文档（本文档体系） |

## 快速开始

```bash
# 启动汇率预测服务
cd fx_system/backend
python3 server.py

# 访问前端
# http://127.0.0.1:8010
```

## 核心流程

```
数据采集 → 数据清洗 → 特征工程 → 模型训练 → 预测输出 → 回测验证 → 预警推送
```

## 注意事项

⚠️ 本系统用于**研究与半自动决策支持**，实盘接入前必须：

1. 完成历史数据滚动窗口 walk-forward 回测
2. 经过模拟盘验证（2-4 周）
3. 设置仓位上限与回撤熔断机制
